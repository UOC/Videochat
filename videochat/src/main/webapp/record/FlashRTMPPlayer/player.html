Webcam Recording

Flash RTMP Player

Description

This example requires a single computer running Adobe Flash in a web browser and a web camera. It illustrates Wowza Streaming Engine's advanced video recording capabilities using the Adobe Flash player.

Note: Webcam Recording cannot be configured using Wowza Streaming Engine Manager, you must perform configuration using the xml files in the Wowza Streaming Engine /conf directory.

Installation

In the /examples/WebcamRecording directory:

  • LINUX
    Run ./installall.sh
  • WINDOWS 7 / WINDOWS 8
    Right-click installall.bat and then select Run as administrator
  • WINDOWS SERVER
    Double-click the installall.bat file
  • MAC OS X
    Double-click the installall.command file

Instructions

  1. Review the following tutorial:
    How to set up live video recording
  2. Click the Connect button.
  3. When the video appears, click the Record button.
  4. After some video has been recorded, click the Stop button.
  5. To playback the video, click the Play button.
  6. To record more video, click the Record button again. If you want to append the new video to the previous recording, select the Append box.
