Video Chat

Flash RTMP Player

Description

This example illustrates Wowza Streaming Engine video chat capabilities using the Adobe Flash player.

Installation

In the /examples/VideoChat directory:

  • LINUX
    Run ./installall.sh
  • WINDOWS 7 / WINDOWS 8
    Right-click installall.bat and then select Run as administrator
  • WINDOWS SERVER
    Double-click the installall.bat file
  • MAC OS X
    Double-click the installall.command file

Instructions

  1. Review the following tutorial:
    How to set-up live video chat.
  2. Mute the audio on your computer to prevent feedback.
  3. Click the Connect button.
  4. When the video appears on the left, click the Publish button. This video is the video sender.
  5. To receive the video, click the Play button on the right.
